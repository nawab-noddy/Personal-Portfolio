# Mohammad Anas - Personal Portfolio


This is my personal portfolio website, built from scratch to showcase my skills, professional experience, and personal projects. It features a modern, fully responsive dark theme with several dynamic and interactive elements to create an engaging user experience.

## âœ¨ Key Features

* **Fully Responsive:** The design adapts seamlessly to all device sizes, from mobile phones to desktop monitors.
* **Interactive Animated Background:** A "live" background with subtle, fast-moving shapes that react to your mouse movement with a parallax effect.
* **Dynamic UI Elements:**
    * **Typed.js:** A "typing" animation in the hero section.
    * **Scroll-to-Top Button:** Appears on scroll for easy navigation.
    * **Scroll Progress Bar:** Shows the reading progress at the top of the page.
    * **Reveal-on-Scroll:** Content sections (About, Skills, etc.) fade in as you scroll down.
* **Modern Dark Theme:** A clean, professional dark mode with a purple accent color.
* **CSS Animations:**
    * Constant "float" animation on the "About Me" image.
    * Hover effects on all links, buttons, and cards (including zoom-on-hover for project images).
    * Animated underline for navbar links.

## ðŸ’» Tech Stack

* **HTML5:** For the core structure and content.
* **CSS3:** For all styling, including:
    * Flexbox & Grid
    * Media Queries (for responsive design)
    * Keyframe Animations
    * Transitions & Transforms
    * `z-index` stacking for layered backgrounds
* **JavaScript (ES6+):** To power all interactive features, including:
    * Mobile menu toggle
    * Header-on-scroll effects
    * Scroll-to-top button logic
    * Interactive mouse-move (parallax) background
* **Typed.js:** For the hero section's typing animation.
